Creating a boot function

To create a boot function you will have to create a new .js file inside the functions/boot folder.
This is what a default boot function looks like. 


exports.name = "name";
exports.time = 0
exports.delay = 0;

exports.function = function(client) {

  // execute your code on boot here.

}

Possible exports

  • .name
    TypeString
    Required

    Describes the name of the message function. Must be unique

  • .time
    TypeNumber / null
    Default0

    time before a command will be re-executed, this number is in milliseconds. Use 0 to disable.

  • .delay
    TypeNumber / null
    Default0

    How much delay there will be before this boot function will run, starts running after ready event is fired. This number is in milliseconds. Use 0 to disable.

  • .function
    Typefunction
    Required

    Code that will be executed before a command runs.
