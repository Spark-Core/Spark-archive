Snippets

Snippets are functions that can be used throughout your whole code.


Creating snippets

exports.name = "name"
exports.function = function(){

// Use this as a regular function

}

Possible exports

  • .name
    TypeString
    Required

    Describes the name of the snippet. Must be unique

  • .function
    Typefunction
    Required

    The function that will run when you use the snippet.

    • Using snipppets

    To use a snippet in your code use it like this:

    client.snippets.name()

    To get a list of your snippets, use this:

    client.snippets.list()
