Creating a message function

To create a message function you will have to create a new .js file inside the functions/messages folder.
This is what a default message function looks like. 


exports.name = "name-mf"
exports.type = ["commands", "messages"];
exports.function = function(client, message, command) {

  // execute your code here

}

Possible exports

  • .name
    TypeString
    Required

    Describes the name of the message function. Must be unique

  • .type
    TypeArray(string) / null
    Default["messages"]

    The type of the message function that will allow it to be triggered before a message, command or both.

    • message | Only messages
    • command | Only commands
    • all | Both messages and commands

  • .function
    Typefunction
    Required

    Code that will be executed before a command runs.

    • Stopping commands from executing

    To stop a command from executing return true to make it stop executing, or false to let it continue.
